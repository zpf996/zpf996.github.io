# 动态规划（dp）详解以及leetcode经典例题（总）
## 动态规划
```
动态规划是以局部最优解最终得全局最优解。
通俗讲将一个大问题分解为许多小问题，并且每个小问题都求最优解，以得出这个大问题的最优解。
在设计动态规划算法时，需要确认原问题与子问题、动态规划状态、边界状态结值、状态转移方程等关键要素。
```
